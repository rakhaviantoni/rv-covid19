# rv-covid19

- Get worldwide Coronavirus disease (COVID-19) reporting
- Active daily reporting of your country's COVID-19 statistics
- Data: Country, Cases, Deaths, Recovered, Active, Critical
- Sort: `Cases`, `Today Cases`, `Deaths`, `Today Deaths`, `Recovered`, `Active`, `Critical`

<br>

## Install

```sh
# Install with yarn (recommended).
yarn install

# Run directly with yarn.
yarn start
```
